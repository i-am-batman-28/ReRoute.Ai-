FLIGHT STATUS & BASICS
--------------------------------------------------------------------------------
AviationStack
  https://aviationstack.com
  Free: small monthly quota — good for demos; watch rate limits.

OpenSky Network
  https://opensky-network.org
  Free: registration — live ADS-B / state vectors; research-style demos.

Aviation Edge
  https://aviation-edge.com
  Free: limited tier — check current limits for prototyping.


FLIGHT SEARCH / ALTERNATIVES (SANDBOX / TEST)
--------------------------------------------------------------------------------
Amadeus Self-Service (for Developers)
  https://developers.amadeus.com
  Free: test/sandbox — classic hackathon choice; not real booking.

Duffel
  https://duffel.com
  Free: test mode — offer-style flows in test.

Kiwi / Tequila (check current policy)
  https://docs.kiwi.com
  Free: historically limited dev tier — confirm current terms.


WEATHER (DELAY RISK / CONTEXT)
--------------------------------------------------------------------------------
Open-Meteo
  https://open-meteo.com
  Free: no API key — forecasts + historical; fair-use friendly.

OpenWeatherMap
  https://openweathermap.org/api
  Free: tier with API key — daily limits on free plan.

National Weather Service (US)
  https://www.weather.gov/documentation/services-web-api
  Free: US-centric.


MAPS & MULTI-MODAL ROUTING (GOOGLE ALTERNATIVES)
--------------------------------------------------------------------------------
OpenRouteService
  https://openrouteservice.org
  Free: tier with API key — directions, isochrones, etc.

OSRM (Open Source Routing Machine)
  http://project-osrm.org
  Free: public demo servers (light use) or self-host locally.

Nominatim (OpenStreetMap geocoding)
  https://nominatim.org
  Free: respect OpenStreetMap usage policy — no heavy scraping.

Google Maps Platform
  https://developers.google.com/maps
  Note: $200/month credit — billing account usually required; not zero-card “free.”


NOTIFICATIONS
--------------------------------------------------------------------------------
Firebase Cloud Messaging (FCM)
  https://firebase.google.com/docs/cloud-messaging
  Free: generous tier — push notifications.

Twilio
  https://www.twilio.com
  Note: trial credits — fine for demo SMS; not long-term free.


VOICE (BROWSER)
--------------------------------------------------------------------------------
Web Speech API
  https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API
  Free: built into Chromium browsers — no vendor key; quality varies by device.


LLM / AGENT (“BRAIN”)
--------------------------------------------------------------------------------
OpenAI / Anthropic / similar
  Note: typically paid; hackathons may use credits or small budget.

Ollama + local open models
  https://ollama.com
  Free: runs on your machine — $0 API cost; heavier setup.


EMAIL (OPTIONAL)
--------------------------------------------------------------------------------
Resend
  https://resend.com
  Free: tier with limits.

SendGrid
  https://sendgrid.com
  Free: tier with limits.


MINIMAL “ALL FREE / NO CARD” STACK (SOLID DEMO)
--------------------------------------------------------------------------------
1. Open-Meteo — weather
2. AviationStack (or OpenSky + custom UI) — flight-related data
3. Amadeus sandbox — alternative “flights” in test
4. OpenRouteService + Nominatim — routes / places (respect limits)
5. FCM — push (if you add mobile/web push)
6. Web Speech API — voice
7. Mock/cached JSON in your backend — when quotas bite (valid for hackathon)
